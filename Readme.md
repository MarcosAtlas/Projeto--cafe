# Roda Coffee

Bem-vindo ao **Roda Coffee**, um site para promover o melhor café da região, com um ambiente agradável e aconchegante.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

```
assets/
    css/
        

style.css


    img/        


index.html


```

### Arquivos principais

- **`index.html`**: Contém a estrutura HTML do site, incluindo as seções de cabeçalho, menu, avaliações e endereço.
- **`assets/css/style.css`**: Define o estilo visual do site, como cores, fontes, espaçamentos e layout.

## Funcionalidades

### Seções do Site

1. **Home**: Apresenta o slogan do café e um botão para acessar o menu.
2. **Sobre**: Explica o que torna o café especial, com uma imagem e descrição.
3. **Menu**: Mostra os itens disponíveis no cardápio, com imagens, descrições e preços.
4. **Avaliações**: Exibe depoimentos de clientes satisfeitos, com fotos e avaliações.
5. **Endereço**: Inclui um mapa interativo para localizar o café.

### Estilos Principais

- **Cores**: Utiliza variáveis CSS para definir cores principais, como `--primary-color` (#d3ad7f) e `--bg` (#010103).
- **Fonte**: Usa a fonte `'Roboto', sans-serif` para um design moderno e limpo.
- **Responsividade**: O layout é ajustado para diferentes tamanhos de tela usando `grid` e `flexbox`.

## Como Usar

1. Clone este repositório:
   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```
2. Abra o arquivo `index.html` em qualquer navegador para visualizar o site.
